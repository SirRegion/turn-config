# Powershell Module for MdctecMaintenanceMenu

This is a Windows PowerShell script module providing a console-based menu to execute several maintenance tasks related to complianceBase setup and infrastructure.


## Installation

1)  Extract the powershell-modules.zip to any destination folder (e.g. Desktop Folder)

2)  Open Windows PowerShell as Administrator, navigate to the extracted folder and execute the file `./setup.ps1`
    This will register the MdctecMaintenanceMenu Powershell Command to the system.

3)  Now you can launch the menu with `MdctecMaintenanceMenu` or its alias `MMM`
